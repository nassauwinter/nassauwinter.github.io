Posting the website to Github Pages

Generating a static website was an easy task. Thanks Pelican. The output is a folder with a bunch of html and css files. All you need to make this a website is to create a public GitHub repo named your_username.github.io and push the output files into main branch there. The resulted website will be available at https://your_username.github.io. More details on GitHub Pages here: https://pages.github.com/

The first website template is a bit ugly though it is simple and understandable. So I'm gonna create my own template. It means to make blog even more uglier :D
